User Management System

Welcome to the User Management System . Below are the available API endpoints:

Authentication Endpoints

  • Register User: POST /api/register/
  • Login (Get Token): POST /api/token/
  • Refresh Token: POST /api/token/refresh/
  • Logout: POST /api/logout/

User Management Endpoints

  • List Users: GET /api/users/
  • User Details: GET /api/users/{id}/
  • Update User: PUT /api/users/{id}/
  • Delete User: DELETE /api/users/{id}/

Authentication Required

All endpoints except Register and Login require authentication. Include the following header with your requests:

Authorization: Bearer <your_access_token>
